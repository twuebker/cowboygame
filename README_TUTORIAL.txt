Cowboy Game


Summary: You are a Cowboy in the Wild West. That means you have a Colt of course and you know how to use it. For some reason, the town you are in is full of monsters - the number even increases with every day that goes by.
Go kill them for the Highscore!

Contents: Game features 3 Enemies with different functionalities (Coyote, Cactus, Coffin), 8-way-animated Player, Day-And-Night cycle, Dynamic Lighting, Sound effects, different music at night and common GUI Elements like Main Menu, Pause Menu, Score and Daytime.

Controls:
WASD/Arrows: Move
Left-click: Shoot
Esc: Bring up Pause Menu
